Report tool for no invasive stress physiological analysis

  • Data Analytics
  • Python
  • Bluetooth programming

The tool allows the analysis of physiological data for participants in virtual reality sessions. The parameters detected are useful for a posteriori analysis, allowing to evaluate the patient's state during the setting, relaxation and interaction phase. Other parameters such as liking, personal involvement, relaxation and interaction are evaluated by the health care professional/technician during the test run. At the end of the protocol a PDF report is produced to document the activity performed by that specific patient.
